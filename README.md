# StripePy

## Install instructions

```bash
python3 -m venv venv
venv/bin/pip install . -v
venv/bin/stripepy --help
```

## Build instructions

```bash
python3 -m venv venv
venv/bin/pip install --upgrade pip
venv/bin/pip install build
venv/bin/python -m build
```
